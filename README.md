# volVIC

**VOLume-based Virtual Image Correlation (volVIC) library for Python**
Python library for isogeometric multipatch surface fitting of volumetric images, based on the method described in [Bichet et al., 2025](https://doi.org/10.1016/j.cma.2025.117729).
